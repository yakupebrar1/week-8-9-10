> Inspired by [CS50's Web Track](https://cs50.harvard.edu/x/) · Practice exercises for learning frontend fundamentals

## 📚 Overview

This week focuses on the three pillars of frontend web development:

- **HTML** — Structure and semantics
- **CSS** — Styling, layout, and responsiveness
- **JavaScript** — Interactivity and dynamic behavior

By the end of these exercises, you'll be able to build complete, interactive web pages from scratch.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- A text editor ([VS Code](https://code.visualstudio.com/) recommended)
- Basic understanding of how the web works (HTTP, browsers)

### How to use this repo

1. **Clone** the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/week9.git
   cd week9
   ```

2. **Navigate** to an exercise folder:
   ```bash
   cd exercise1-html-basics
   ```

3. **Read** the exercise `README.md` inside each folder.

4. **Open** `index.html` in your browser (just double-click, or use Live Server in VS Code).

5. **Edit** the files to complete the tasks. Each exercise has a `solution/` folder you can peek at if you're stuck.

### Recommended VS Code Extensions
- **Live Server** — Auto-refreshes browser on save
- **Prettier** — Code formatter
- **HTML CSS Support** — Better autocomplete

## 🧠 How to Succeed

- **Don't skip exercises** — each one builds on the last
- **Type the code yourself** — don't copy-paste; muscle memory matters
- **Read error messages** — the browser console is your friend (F12)
- **Experiment** — change values, break things, learn why
- **Use MDN** — it's the gold standard reference for web APIs
