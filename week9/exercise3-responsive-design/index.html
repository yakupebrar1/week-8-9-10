About Responsive Design

Responsive web design is an approach where a website's layout adapts to the screen size and orientation of the device being used to view it.

The three pillars of responsive design are: flexible grids, flexible media, and media queries. Together they allow a single HTML document to look great on anything from a smartwatch to a 4K monitor.
