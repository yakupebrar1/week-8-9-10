# Mini Projects

Four graded in-class projects with increasing difficulty.  
Complete them in order during class sessions using the peer mentoring workflow.

---

## Project Overview

| # | Title | Difficulty | Time | Group Size |
|---|-------|-----------|------|-----------|
| 1 | Language Poll Analyser | 🟢 Easy | 20–25 min | Pair |
| 2 | Grade Tracker | 🟢 Easy | 25–30 min | Pair |
| 3 | Event Log Reporter | 🟡 Intermediate | 35–45 min | 3 people |
| 4 | Survey Dashboard | 🔴 Hard | 60–90 min | 4 people |

---

## How to Submit Each Project

1. Create your branch from your group branch:
   ```bash
   git checkout group/your-team
   git pull origin group/your-team
   git checkout -b yourname/project1
   ```
2. Work inside the project folder only — do not touch other folders.
3. When complete, push and open a PR into your group branch.
4. Fill in **every section** of the PR template — especially the Teach-Back.
5. Roles rotate every project (see the role rotation table in the root README).

---

## Role Reminder

| Role | Core duty this session |
|------|----------------------|
| 🧑‍💻 Coder | Types the code; narrates every line out loud |
| 🔍 Reviewer | Asks "why?" for every decision; leaves 2+ PR comments |
| 🧪 Tester | Creates test inputs; pastes output as PR evidence |
| 🏫 Mentor | Writes the Teach-Back; ensures everyone contributes |
